# Kuhnert Autoverwertung Website

Statische Onepage-Website für Autoverwertung O. Kuhnert GmbH & Co. KG.

## Lokal testen

```bash
python3 -m http.server 8020
```

Danach `http://127.0.0.1:8020/index.html` öffnen.

## Veröffentlichung

Die Seite besteht aus statischen HTML-, CSS-, JavaScript- und Bilddateien und kann direkt über Vercel oder klassischen Webspace veröffentlicht werden.
